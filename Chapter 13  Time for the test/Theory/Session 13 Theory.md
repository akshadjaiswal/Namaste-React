## Setting Up testing in our app
- Install React Testing Library
- Install Jest
- Install Babel Dependencies
- Configure Babel
- Configure Parcel Config file and disable default babel transpilation
- Configure JEST jest --init
- Installed JSDOM Library 
- Testing jest
